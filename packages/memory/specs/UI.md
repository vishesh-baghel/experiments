## Design System

See **[DESIGN.md](./DESIGN.md)** for the complete design system, including:

- **Typography**: Lora (headings), IBM Plex Sans (body), IBM Plex Mono (code)
- **Colors**: Dark-first palette inspired by [zed.dev](https://zed.dev)
- **Components**: Buttons, cards, inputs, tags, sidebar styles
- **Animations**: Subtle transitions and loading states
- **Markdown rendering**: Prose styles for document content
